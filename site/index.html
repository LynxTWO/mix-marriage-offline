The MMO workflow

Analyze once. Render everywhere.

01

Scan stems

Point MMO at your stem folder. It validates formats, measures levels, and builds a deterministic issue map.

→
02

Plan renders

One planning pass targets stereo, 5.1, 7.1, 7.1.4, and any mix of the five layout standards.

→
03

Ship confidently

Installer artifacts, checksums, signatures, translation QA, and PDF recall sheets in one run.

Terminal 
# Scan + render to all 5 layout standards in one pass
$ mmo safe-render \
    --report out/report.json \
    --render-many \
    --render-many-targets stereo,5.1,7.1.4 \
    --out-dir out/deliverables/

# Watch folder: deterministic batch on every stem drop
$ mmo watch ./stems --out-dir out/ --render-many

# Browse the offline plugin marketplace
$ mmo plugin list
